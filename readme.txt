Integrates Connex Mojo middleware into WordPress.

== Description ==

This plugin integrates Connex Mojo middleware into WordPress, providing various shortcodes for displaying members, events, and committees.

== Installation ==

1. Upload the `connex-mojo-integration` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
